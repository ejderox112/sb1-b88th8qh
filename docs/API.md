# API Uçları

## Supporters
- GET /supporters: Kullanıcının bağışlarını listeler
- POST /supporters: Yeni bağış ekler

## Supporter Badges
- GET /supporter_badges: Kullanıcının rozetlerini listeler

## User Reports
- GET /user_reports: Kullanıcının raporlarını listeler
- POST /user_reports: Yeni rapor ekler

## Moderation Actions
- GET /moderation_actions: Moderatör aksiyonlarını listeler
- POST /moderation_actions: Yeni aksiyon ekler
