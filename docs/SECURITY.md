# Güvenlik

- Supabase RLS (Row Level Security) tüm ana tablolarda aktif
- Her kullanıcı sadece kendi verisini görebilir/ekleyebilir

- API anahtarları ve gizli bilgiler .env dosyasında tutulmalı
- Otomatik test ve CI/CD ile güvenlik açıkları erken tespit edilir
