# TESTING

Tüm modüller için test senaryoları ve validasyon adımları:

- Indoor katkı/moderasyon: Supabase entegrasyonu, submit/approve/reject testleri
- POI öneri/onay: Supabase tablosu, öneri ekleme/onaylama/ret testleri
- Grup görevleri: XP dağıtımı, tamamlanma, backend fonksiyonları
- Task tamamlama: XP ve rozet/ödül entegrasyonu
- Badge/ödül: Hesaplama, dağıtım, validasyon
- Flag/report/moderation: UI ve backend entegrasyonu, rapor/aksiyon testleri

Her modül için Jest ve demo kullanıcı akışı ile testler tamamlanacaktır.
